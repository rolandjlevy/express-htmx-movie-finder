# Htmx Movie Finder

A showcase of HTMX in action — fast, elegant, server‑driven UI updates without the overhead of a client‑side framework.

### Live demo hosted on Vercel

- [Movie Finder Demo](https://express-htmx-movie-finder.vercel.app/)

### View the project in VS Code Simple browser

- Get the URL for the page - see PORTS in terminal
- From command palette: Simple Browser: Show
- then paste in that URL

### To run this project in a Codespace

- duplicate `.env-example` and rename it to `.env` then add a valid OMDB API key to it
- in the Terminal run:

```bash
npm i -g http-server
http-server -c-1
```

- then click 'Open in Browser'
